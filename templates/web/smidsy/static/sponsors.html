[% loc('The journey so far') %]

With support from our sponsors we have been able to complete this initial version of Collideoscope: a site that collects both incident and near miss data in order to present a clearer picture of cycle safety in the UK

[% loc('Our roadmap') %]

We are currently focused on the launch of Collideoscope, but intend to add a number of additional features to the site if we can attract more sponsors. In the future we would like to add notifications to the relevant police department, add incident-aware route planning and create a mobile application to offer advice to someone involved in an incident.
